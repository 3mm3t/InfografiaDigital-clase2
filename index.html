Gráficos figurativos, no figurativos y mixtos

Emmet Chavez Flood & Constanza Alvarado

Los gráficos son una visualización de información, dependiendo el tipo de información que se quiere exponer, se van a dividir entre los siguientes graficos:

GRÁFICA FIGURATIVA

Es informacion que representan la realidad a traves de imagenes.

GRÁFICA MIXTA

Este tipo de graficos, es una mezcla de grafico figurativo y no figurativo.

GRÁFICA NO FIGURATIVA

Son la visualizacion de los datos, nos permiten dar forma algo que no existe en la realidad.

La gráfica de arriba se despliega en la página como si se tratara de una imagen rasterizada, con el elemento <img>. Otros elementos de HTML podrían usarse para un mejor despliegue del SVG. Reviseramos tales elementos algunas clases más adelante, pero ya les conviene revisar Using SVG | CSS Trick.
